# Database Scripts & Schema (`database/`)

This directory maintains the relational SQL schemas, database setup queries, seeds, and migration scripts.

## Structure Overview
- **[schema.sql](file:///D:/PROJECT/track_daily/database/schema.sql)**: Complete database schema including tables, keys, triggers, indices, and constraints.
- **[seed.sql](file:///D:/PROJECT/track_daily/database/seed.sql)**: Seed values to prepopulate tables during development.
- **[reset.sql](file:///D:/PROJECT/track_daily/database/reset.sql)**: Clean drop-tables script for reset/testing.
- **[migrations/](file:///D:/PROJECT/track_daily/database/migrations/)**: Chronological/versioned schema migration files.
- **[queries.sql](file:///D:/PROJECT/track_daily/database/queries.sql)**: Reference SQL queries.
