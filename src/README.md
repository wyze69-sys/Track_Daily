# Frontend React App & Local API Server (`src/`)

This directory contains the frontend user interface, local API server, and file-backed development data layer used by logweb.

## Folder Directory Structure
- **[components/](file:///D:/PROJECT/logweb/src/components/)**: Reusable React components (buttons, cards, layout structures).
- **[pages/](file:///D:/PROJECT/logweb/src/pages/)**: Primary page containers and routing targets (Dashboard, Workout Log, Admin, etc.).
- **[services/](file:///D:/PROJECT/logweb/src/services/)**: API integration clients for the real routes mounted under `/api`.
- **[server/](file:///D:/PROJECT/logweb/src/server/)**: Express API server mounted with Vite middleware during local development.
- **[context/](file:///D:/PROJECT/logweb/src/context/)**: React context providers (auth context, global application state).
- **[routes/](file:///D:/PROJECT/logweb/src/routes/)**: React Router client-side path declarations.
